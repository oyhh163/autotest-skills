## 测试发现

### 百度搜索特性
1. **高亮实现**: 使用 `<em>` 标签，颜色为 `rgb(247, 49, 49)` (红色)
2. **空格处理**: 纯空格输入不会触发搜索跳转，保持在首页
3. **推荐展示**: 首页展示热搜榜，搜索框聚焦后可能展示下拉建议

### 元素选择器（百度）
```javascript
searchInput: '#kw'              // 搜索输入框
searchButton: '#su'             // 百度一下按钮
searchResults: '#content_left'  // 结果区域
resultItem: '.result, .c-container'  // 单个结果
highlightedKeyword: 'em'        // 高亮元素
```
