# 自动化候选分析 - 搜索功能（Web）

## 筛选标准（依据 skill 自动化规则）

1. **P0 核心场景**：主流程必测。
2. **页面文案校验**：以设计/需求为准，覆盖全部文案，动态文案略过转人工。
3. **页面跳转规则**：按钮点击 / 条件跳转，URL 或页面元素断言。
4. **按钮禁用/启用状态**：依据文档逻辑断言。
5. **数据提交/计算**：依据用例预期断言。
6. **异常输入错误提示**：无效输入的提示校验。

## 暂不自动化 / 转人工校验

- **DET-003（推荐内容）**：推荐来源/规则未定，属动态内容 → 人工校验（风险 R2）。
- **DET-005（结果相关性）**：语义相关性难以稳定断言 → 人工校验。
- **DET-007（仅空格判定）**：规则待确认（R1）→ 确认后再自动化。
- **DET-008（匹配粒度/大小写）**：规则待确认（R3）→ 确认后再自动化。
- **DET-009（空结果展示）**：需求未定义（R4）→ 人工校验。
- **DET-010（回车触发）**：是否支持待确认（R5）。
